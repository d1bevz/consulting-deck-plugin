# Visual Metaphors for Slide Design

Five core patterns for organizing information within a slide:

## 1. Linear Logic
- Cause -> Effect
- Action -> Result
- Input -> Process -> Output
- Use when: showing flow, sequence, causation

## 2. Opposing Logic
- Pro vs Contra
- Support vs Resistance
- Internal vs External
- Before vs After
- Use when: showing contrast, trade-offs, comparison of two sides

## 3. Hierarchy
- Parent -> Children
- Decomposition into parts
- Grouping by category
- Use when: showing structure, breakdown, classification

## 4. Temporal Development
- Phase 1 -> Phase 2 -> Phase 3
- Steps of a process
- Milestones on a timeline
- Use when: showing progression, evolution, roadmap

## 5. Comparison / Detail
- Side-by-side tables
- Matrices
- Grids with scores
- Use when: detailed comparison of multiple objects across criteria

## Combination
Real slides are typically **combinations** of these patterns. For example:
- Timeline (temporal) with annotations (linear cause->effect)
- 2x2 matrix (comparison) with quadrant labels (hierarchy)
- Two-column quotes (opposing) with summary (linear conclusion)

## Why This Matters
- Draws audience attention
- Helps process information faster
- Supports structured thinking
- Enables storytelling through visual structure
